# UI Design

The frontend uses a dark glassmorphism theme with:
- Gradient backgrounds
- Subtle hover glows
- Smooth micro‑animations for button clicks and loading spinners
- Google Font **Inter** for modern typography

Key components:
- **ChatTab** – main chat interface, warning banner for missing index.
- **AutoGenTab** – automatic generation panel, extended slider (1‑20 turns), turn‑pair pills.
- **Sidebar** – model selector, settings, and API‑key input.
